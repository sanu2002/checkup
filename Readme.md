Installation

1. Clone the repository:

		git clone https://github.com/sanu2002/checkup
        cd checkup

2.Install dependencies:
		forge install

3. forge build
	forge script script/Deploy.s.sol --fork-url http://127.0.0.1:8545 --private-key <PRIVATE_KEY>
